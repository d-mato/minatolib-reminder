図書館で借りた本の返却日をGoogleカレンダーに反映させるプログラム

- 定期的に図書館サイトにログインし、貸出状況をチェック
- Googleカレンダーに本のタイトルと返却日を登録する
- 返却日の前日にリマインダーメールを送信する
- **港区図書館限定**

これらを処理を行うスクリプトをcronに登録して運用する


### Modules

#### minatolib-client.js
図書館へのログインと貸出状況の確認を行う

#### gcalendar-register.js
Googleカレンダーへ予定を登録する

#### app.js
一連の処理を行うメインスクリプト
